Troubleshooting a PC

  1. Power or display is not present in computer
  2. Computer restarts/shuts down/ hangs after few minutes runtime
  3. Hard disk/ Boot device not found error or Dumping message while booting
  4. IDE/AHCI Mode or Legacy/UEFI Mode selection
  5. BIOS password solutions with Cmos battery/Jumper

    1. No power:

    • Check smps and wiring
    • Check Casing power button
    • Remove extra 4 pin power connector from motherboard and check status Check grey +5V (power good) wire from smps

      No display:

    • Check display port and wire and wiring
    • Check if Num locks are light on keyboard problem is with Monitor Else
    • check Ram and cpu and clean and reassemble them else
    • check SMPS grey (+5V power good) and purple wiring (+5V stand by)
    Generally Incase of faulty graphics card, lining appears Incase of faulty North bridge, display flickers Incase of processor display will hang. Advanced : Use a power supply tester to check if the PSU is providing the correct voltages to the motherboard.

    2. Restarts:

    • Service your PC and Apply good cooling paste on cpu
    • Check if Ram and cpu meet with motherboard s requirements
    • Check cpu fan and smps fan and wire of cpu fan connecting smps
    • Check cpu rpm at BIOS setting and see if it is as mentioned by provider(generally >1600)
    • Check for updates and antivirus software

    3. Boot device:

    • Switch with different ports in motherboard Check the disk with another pc
    • Try replacing SATA or PATA cable
    • Select your hard disk as first priority boot device at BIOS setting
    • South bridge might be faulty Bad sector might be present in hdd

      • Dumping message:

    • Error with white text error code on blue screen
    • Generally caused by faulty hard disk drive or RAM Try servicing them and changing them

    4. Modes:

    • Older pc needs to be in IDE mode at BIOS for os setup
    • Modern pcs can be legacy or uefi Make bootable device in MBR for legacy, and uefi in uefi

    In what the hard disk is formatted in, the bootable device also should be in formatted at that for proper installation

  5. BIOS CMOS Remove cmos battery for over an hour to set default bios configuration Or Shot CMOS jumper for over 10 seconds
